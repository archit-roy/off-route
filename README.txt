Off Route Media website prototype

Open this folder in VS Code.

Required structure:

index.html
styles.css
script.js
assets/
  logo-mark.svg
  logo-with-text.svg

To view it:

1. Open index.html directly in your browser, or
2. In VS Code, install the Live Server extension, right-click index.html, and choose "Open with Live Server".

Do not move index.html away from styles.css, script.js, or the assets folder unless you also update the paths inside index.html.
